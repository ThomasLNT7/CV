Hackescape

Le but du projet

Un escape game en ligne

Hackescape est un site web qui propose des escapes games en ligne avec différents thèmes. Actuellement il y a un escape game de disponible qui est sur le thème de base du site qui est l'informatique et qui vous mets en plein milieu d'un piratage de vos données.

Histoire

Projet scolaire "Création de site web avec CMS"

Ce projet est né d'un des différents cours que je suis à l'ISG qui est "Création de site web avec CMS", qui avait pour finalité la création d'un site web avec le CMS WordPress. On avait le choix sur le type de site, avec mon équipe nous ne voulions pas faire un simple site vitrine, on voulait créer une expérience sur notre site d'où cette idée d'escapegame qui est en vogue de nos jours.

Extrait du projet

Screen du projet

Voici la page d'accueil du site web, je ferais en sorte qu'il soit bientôt accessible en clair facilement.

Difficulté rencontrée

Nouvel outil : Wordpress

La plus grande "difficulté" du projet, était d'apprendre à utiliser l'outil, plus que complet qu'est WordPress. Dans le cas de notre site web, nous n'avons utilisé qu'une infime partie de ce que propose WordPress. C'est un outil très important à aux minima comprendre pour le futur selon moi.

Finalité

Appréciation de la classe et du professeur

Nous avons fait tester notre site à tout notre groupe de classe en même temps lors d'une épreuve type course. Tout le monde s'est amusé !
